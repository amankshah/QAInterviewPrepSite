Basics & Setup

Q1-9
1 What is Playwright?
Playwright is a Node.js library for automating web browsers. It enables reliable end-to-end testing for modern web apps.
2 Which browsers does Playwright support?
Chromium, Firefox, and WebKit (Safari's rendering engine).
3 How do you install Playwright?
npm install @playwright/test
4 How do you launch a browser in Playwright?
const browser = await playwright.chromium.launch();
5 How do you open a new page in Playwright?
const page = await browser.newPage();
6 How do you navigate to a URL in Playwright?
await page.goto('https://example.com');
7 How do you take a screenshot in Playwright?
await page.screenshot({ path: 'screenshot.png' });
8 How do you set the viewport size in Playwright?
await page.setViewportSize({ width: 1280, height: 720 });
9 How do you emulate a mobile device in Playwright?
await page.emulate(devices['iPhone 11']);

Advanced Topics

Q91-101
91 How do you use Playwright with TypeScript?
Install TypeScript and configure Playwright with TypeScript support.
92 How do you integrate Playwright with CI/CD pipelines?
Use Playwright test runners and integrate with CI/CD tools like GitHub Actions, Jenkins, or CircleCI.
93 How do you record video of a test run?
Configure recordVideo in the browser context options.
94 How do you blur an element in Playwright?
await page.blur('selector');
95 How do you handle form submission?
await page.click('form [type="submit"]');
96 How do you check if a checkbox is checked?
const isChecked = await page.isChecked('input[type="checkbox"]');
97 What are Playwright's Actionability Checks?
Playwright automatically checks if an element is Visible, Enabled, and Stable before performing any action. This prevents flaky tests.
98 What is the difference between page.click() and locator.click()?
page.click() is deprecated. locator.click() is the modern approach that provides better auto-waiting and retry logic.
99 What are fixtures in Playwright?
Fixtures are reusable pieces of test setup/teardown. Built-in fixtures include page, context, and browser.
100 How do you create a custom fixture in Playwright?
Use test.extend() to define custom fixtures that can be used across tests.
101 What is Page Object Model (POM) in Playwright?
POM is a design pattern where page elements and actions are encapsulated in classes, making tests more maintainable and reusable.
