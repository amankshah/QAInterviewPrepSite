Selenium Basics

Q1-15
1 What is Selenium?
Selenium is an open-source automation testing framework for web applications. It supports multiple browsers, programming languages, and operating systems.
2 What are the components of Selenium?
Selenium WebDriver: Browser automation API
Selenium IDE: Record and playback tool
Selenium Grid: Parallel test execution on multiple machines
3 What is WebDriver in Selenium?
WebDriver is an interface that allows you to control web browsers programmatically. Each browser has its own driver (ChromeDriver, GeckoDriver, EdgeDriver).
4 How do you set up Selenium WebDriver in Java? 
// Add dependency in pom.xml
<dependency>
    <groupId>org.seleniumhq.selenium</groupId>
    <artifactId>selenium-java</artifactId>
    <version>4.x.x</version>
</dependency>

// Setup driver
WebDriver driver = new ChromeDriver();
driver.get("https://example.com");
5 What browsers does Selenium support?
Chrome, Firefox, Safari, Edge, Opera, and Internet Explorer. Each requires its specific driver.
6 What is the difference between Selenium 3 and Selenium 4?
Selenium 4: - W3C WebDriver standard compliance
- Relative locators
- Better Chrome DevTools support
- Enhanced Selenium Grid
- No need for separate driver executables (WebDriverManager)
7 How do you launch a browser in Selenium? 
WebDriver driver = new ChromeDriver();
driver.get("https://example.com");
driver.navigate().to("https://example.com");
8 What's the difference between get() and navigate().to()?
get(): Waits for page to load completely
navigate().to(): Part of navigation interface, allows back(), forward(), refresh()
9 How do you close browser in Selenium?
driver.close(); - Closes current window
driver.quit(); - Closes all windows and ends session
10 How do you get current URL and title? 
String url = driver.getCurrentUrl();
String title = driver.getTitle();
String pageSource = driver.getPageSource();
11 How do you maximize browser window? 
driver.manage().window().maximize();
driver.manage().window().fullscreen();
driver.manage().window().setSize(new Dimension(1920, 1080));
12 What is WebDriverManager?
A library that automatically downloads and manages browser drivers. No need to manually download executables. 
WebDriverManager.chromedriver().setup();
WebDriver driver = new ChromeDriver();
13 How do you take screenshots in Selenium? 
TakesScreenshot ts = (TakesScreenshot) driver;
File source = ts.getScreenshotAs(OutputType.FILE);
FileUtils.copyFile(source, new File("./screenshot.png"));
14 How do you handle alerts in Selenium? 
Alert alert = driver.switchTo().alert();
alert.accept();    // Click OK
alert.dismiss();   // Click Cancel
String text = alert.getText();
alert.sendKeys("text"); // For prompt
15 What are limitations of Selenium?
- Cannot automate desktop applications
- Cannot read/write to Excel directly
- No built-in reporting
- Image comparison not supported
- Captcha cannot be automated

Locators & Element Identification

Q16-30
16 What are the locator strategies in Selenium?
1. id - Most preferred
2. name
3. className
4. tagName
5. linkText
6. partialLinkText
7. cssSelector
8. xpath
17 How do you find elements in Selenium? 
// Single element
WebElement element = driver.findElement(By.id("username"));

// Multiple elements
List<WebElement> links = driver.findElements(By.tagName("a"));
18 What's the difference between findElement() and findElements()?
findElement(): Returns first matching element, throws exception if not found
findElements(): Returns list of all matching elements, returns empty list if none found
19 How do you locate by ID? 
driver.findElement(By.id("username")).sendKeys("admin");
20 How do you write CSS Selectors? 
// ID: #username
driver.findElement(By.cssSelector("#username"));

// Class: .btn-primary
driver.findElement(By.cssSelector(".btn-primary"));

// Attribute: input[name='email']
driver.findElement(By.cssSelector("input[name='email']"));

// Contains: input[id*='user']
driver.findElement(By.cssSelector("input[id*='user']"));
21 How do you write XPath? 
// Absolute: /html/body/div/input
// Relative: //input[@id='username']

// Text: //button[text()='Submit']
// Contains: //input[contains(@name,'user')]
// Starts-with: //input[starts-with(@id,'user')]

// Parent: //div[@id='parent']//input
// Following-sibling: //label[@id='user']//following-sibling::input
22 What are Relative Locators in Selenium 4? 
// Above
driver.findElement(RelativeLocator.with(By.tagName("input"))
    .above(passwordField));

// Below
driver.findElement(RelativeLocator.with(By.tagName("button"))
    .below(emailField));

// toLeftOf, toRightOf, near
23 CSS Selector vs XPath - which is better?
CSS: Faster, more readable, doesn't navigate backward
XPath: More powerful, can navigate backward, can use text
24 How do you handle dynamic elements?
- Use contains(), starts-with() in XPath
- Use partial attribute values
- Locate by text
- Use indexes carefully
- Wait for element visibility
25 How do you locate elements in a table? 
// Specific cell
//table[@id='data']//tr[2]//td[3]

// All rows
List<WebElement> rows = driver.findElements(
    By.xpath("//table[@id='data']//tr"));
26 How do you get element text and attributes? 
String text = element.getText();
String value = element.getAttribute("value");
String cssValue = element.getCssValue("color");
boolean displayed = element.isDisplayed();
27 How do you check if element is present? 
public boolean isElementPresent(By locator) {
    try {
        driver.findElement(locator);
        return true;
    } catch (NoSuchElementException e) {
        return false;
    }
}
28 How do you locate hidden elements?
Hidden elements can be located but cannot be interacted with directly. Use JavaScriptExecutor to interact.
29 What is StaleElementReferenceException?
Thrown when element is no longer in DOM (page refreshed, DOM changed). Solution: Re-locate the element.
30 How do you handle NoSuchElementException?
- Use correct locator
- Add explicit wait
- Check if element is in iframe
- Verify element is present in DOM

Waits & Synchronization

Q31-40
31 What are the types of waits in Selenium?
1. Implicit Wait: Global wait for all elements
2. Explicit Wait: Wait for specific condition
3. Fluent Wait: Wait with polling interval
32 How do you implement Implicit Wait? 
driver.manage().timeouts().implicitlyWait(Duration.ofSeconds(10));
Applied globally to all findElement() calls.
33 How do you implement Explicit Wait? 
WebDriverWait wait = new WebDriverWait(driver, Duration.ofSeconds(10));
WebElement element = wait.until(
    ExpectedConditions.visibilityOfElementLocated(By.id("username"))
);
34 What are common ExpectedConditions?
visibilityOfElementLocated()
elementToBeClickable()
presenceOfElementLocated()
invisibilityOfElement()
alertIsPresent()
textToBePresentInElement()
titleContains()
35 How do you implement Fluent Wait? 
Wait<WebDriver> wait = new FluentWait<>(driver)
    .withTimeout(Duration.ofSeconds(30))
    .pollingEvery(Duration.ofSeconds(2))
    .ignoring(NoSuchElementException.class);

WebElement element = wait.until(driver -> {
    return driver.findElement(By.id("dynamic"));
});
36 Implicit vs Explicit Wait?
Implicit: Global, applied to all elements, less flexible
Explicit: Specific condition, more control, recommended
37 How do you wait for page load? 
driver.manage().timeouts().pageLoadTimeout(Duration.ofSeconds(30));

// Or wait for document ready state
new WebDriverWait(driver, Duration.ofSeconds(10))
    .until(driver -> ((JavascriptExecutor) driver)
    .executeScript("return document.readyState").equals("complete"));
38 How do you wait for an element to be clickable? 
WebDriverWait wait = new WebDriverWait(driver, Duration.ofSeconds(10));
WebElement button = wait.until(
    ExpectedConditions.elementToBeClickable(By.id("submit"))
);
button.click();
39 Should you use Thread.sleep()?
NO! Thread.sleep() is a hard wait that blocks execution. Always use WebDriver waits for better reliability and performance.
40 How do you handle TimeoutException? 
try {
    WebDriverWait wait = new WebDriverWait(driver, Duration.ofSeconds(5));
    wait.until(ExpectedConditions.visibilityOfElementLocated(By.id("element")));
} catch (TimeoutException e) {
    System.out.println("Element not found within timeout");
}

Frameworks & Best Practices

Q61-75
61 What is TestNG?
Testing framework inspired by JUnit that provides annotations, parallel execution, test prioritization, grouping, and reporting.
62 What are TestNG annotations?
@Test - Test method
@BeforeMethod / @AfterMethod
@BeforeClass / @AfterClass
@BeforeSuite / @AfterSuite
@DataProvider - Data-driven testing
@Parameters - Parameterization
63 What is Page Object Model (POM)?
Design pattern where each page is represented as a class. Page elements and actions are encapsulated, making tests more maintainable. 
public class LoginPage {
    WebDriver driver;
    
    @FindBy(id="username")
    WebElement usernameField;
    
    @FindBy(id="password")
    WebElement passwordField;
    
    @FindBy(id="submit")
    WebElement loginButton;
    
    public LoginPage(WebDriver driver) {
        this.driver = driver;
        PageFactory.initElements(driver, this);
    }
    
    public void login(String user, String pass) {
        usernameField.sendKeys(user);
        passwordField.sendKeys(pass);
        loginButton.click();
    }
}
64 What is PageFactory?
Class that supports POM by initializing WebElements using @FindBy annotations. 
PageFactory.initElements(driver, this);
65 What is Data-Driven Testing?
Testing where test data is separated from test scripts. Data is read from external sources (Excel, CSV, DB, JSON). 
@DataProvider(name = "loginData")
public Object[][] getData() {
    return new Object[][] {
        {"user1", "pass1"},
        {"user2", "pass2"}
    };
}

@Test(dataProvider = "loginData")
public void loginTest(String username, String password) {
    // Test code
}

67 What is Selenium Grid?
Tool for parallel test execution across multiple machines, browsers, and OS. Has Hub (central point) and Nodes (execution machines).
68 How do you run tests in parallel?
In testng.xml: 
<suite name="Suite" parallel="tests" thread-count="3">
    <test name="Test1">
        <classes>
            <class name="TestClass1"/>
        </classes>
    </test>
</suite>
69 What is Extent Reports?
Popular HTML reporting library for test results with rich UI, charts, and screenshots.
70 What is Maven?
Build automation and dependency management tool. Uses pom.xml to manage project dependencies and build lifecycle.
71 What is Jenkins?
CI/CD tool for automating builds and tests. Selenium tests can be integrated with Jenkins for scheduled/triggered execution.
72 What is Cucumber?
BDD (Behavior Driven Development) tool that uses Gherkin syntax for writing test scenarios in plain English (Given-When-Then).
73 What are Selenium best practices?
- Use Page Object Model
- Use explicit waits over implicit
- Avoid Thread.sleep()
- Use unique and stable locators
- Take screenshots on failure
- Use data-driven approach
- Implement proper exception handling
- Use TestNG/JUnit for test management

77 Write code to handle broken links 
List<WebElement> links = driver.findElements(By.tagName("a"));

for (WebElement link : links) {
    String url = link.getAttribute("href");
    if (url != null && !url.isEmpty()) {
        try {
            HttpURLConnection connection = (HttpURLConnection) 
                new URL(url).openConnection();
            connection.setRequestMethod("HEAD");
            connection.connect();
            int responseCode = connection.getResponseCode();
            
            if (responseCode >= 400) {
                System.out.println("Broken: " + url);
            }
        } catch (Exception e) {
            System.out.println("Error checking: " + url);
        }
    }
}

public void captureScreenshot(String testName) {
    TakesScreenshot ts = (TakesScreenshot) driver;
    File source = ts.getScreenshotAs(OutputType.FILE);
    String destination = "./screenshots/" + testName + "_" + 
        System.currentTimeMillis() + ".png";
    try {
        FileUtils.copyFile(source, new File(destination));
    } catch (IOException e) {
        e.printStackTrace();
    }
}

@AfterMethod
public void tearDown(ITestResult result) {
    if (result.getStatus() == ITestResult.FAILURE) {
        captureScreenshot(result.getName());
    }
}

85 Write code for synchronization utility 
public class WaitHelper {
    WebDriver driver;
    WebDriverWait wait;
    
    public WaitHelper(WebDriver driver) {
        this.driver = driver;
        this.wait = new WebDriverWait(driver, Duration.ofSeconds(10));
    }
    
    public WebElement waitForElement(By locator) {
        return wait.until(
            ExpectedConditions.visibilityOfElementLocated(locator)
        );
    }
    
    public void waitForElementToBeClickable(By locator) {
        wait.until(ExpectedConditions.elementToBeClickable(locator));
    }
    
    public void waitForPageLoad() {
        wait.until(driver -> ((JavascriptExecutor) driver)
            .executeScript("return document.readyState").equals("complete"));
    }
}
